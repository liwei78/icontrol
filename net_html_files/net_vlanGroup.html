Associated Arrays and Lists

Property NameTypeDefault ValueRequiredAccessDescription
membersstring optionalread/writeThe names of the VLANs you want to add to or remove from the VLAN group.
proxyExcludesstring optionalread/writeSpecifies the IP addresses that you want to include in the proxy ARP exclusion list. If you use VLAN groups, you must configure a proxy ARP forwarding exclusion list. F5 recommends that you configure this feature if you use VLAN groups with a redundant system. The reason is that both units need to communicate directly with their gateways and the back-end nodes. Creating a proxy ARP exclusion list prevents traffic from being proxied through the active unit due to proxy ARP. This traffic needs to be sent directly to the destination, not proxied.
