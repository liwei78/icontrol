hoststring optionalread/writeSpecifies the IP address of a remote server to which syslog sends messages. The default value is none.
localIpstring optionalread/writeSpecifies the IP address of the interface that the syslog-ng utility binds with in order to log messages to a remote host. For example, if you want the syslog-ng utility to log messages to a remote host that is connected to a VLAN, you set this parameter to the self IP address of the VLAN.
remotePortinteger optionalread/writeSpecifies the port number of a remote server to which syslog sends messages. The default value is 514.
