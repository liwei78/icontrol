mgmt/tm/gtm/distributedApp/disabledContexts

Adds context-disabled to a specified physical component, or deletes context-disabled from the component. When you add this option to a component, the system does not send application traffic to that component. To re-enable application traffic to the physical component, you delete the context-disabled option from the configuration of the component.
