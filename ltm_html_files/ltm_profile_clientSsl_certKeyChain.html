mgmt/tm/ltm/profile/clientSsl/certKeyChain

Manage the set of certificate, key, passphrase and chain associated with a client SSL profile.

This object has the following members.

Properties

NameTypeDefault ValueRequiredAccessDescription

certstring optionalread/writeSpecifies the name of the certificate installed on the traffic management system for the purpose of terminating or initiating an SSL connection.

keystring optionalread/writeSpecifies the name of a key file that you generated and installed on the system. The default key name is default.key.
ocspStaplingParamsstring optionalread/writeSpecifies the OCSP Stapling Parameters object associated with this cert-key-chain object in a clientssl profile.
passphrasestring optionalread/writeSpecifies the key passphrase if required.
