NameTypeDefault ValueRequiredAccessDescription
blockIdleTimeoutinteger3600optionalread/writeConfigures the time after the last connection using the block is freed that the block assignment expires. The default value is 3600 seconds.

zombieTimeoutinteger0optionalread/writeConfigures the timeout after which connections using the zombie block are killed. After connections are killed zombie block is freed after port-block-allocation.block-idle-timeout. This parameter is unused unless the port-block-allocation.block-lifetime is set. The default value is 0 which corresponds to infinite timeout.
